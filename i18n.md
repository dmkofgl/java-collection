# Internationalization (i18n)

Java provides a robust Internationalization (i18n) framework through `java.util` and `java.text` packages.

- `Locale` — language + region (e.g., `en_US`).
- `ResourceBundle` — localized resources.
- `MessageFormat` — parameterized messages.
- `NumberFormat`, `DateFormat`, `Collator` — locale-sensitive formatting and comparison.

Example

```java
Locale locale = new Locale("fr", "FR");
ResourceBundle bundle = ResourceBundle.getBundle("messages", locale);
String greeting = bundle.getString("greeting");
```

File structure for resource bundles:
- `messages.properties` (default)
- `messages_en.properties`
- `messages_fr.properties`

Notes: prefer locale-aware formatting and comparisons when building user-facing features.
